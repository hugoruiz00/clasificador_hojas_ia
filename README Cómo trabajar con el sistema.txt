En primer lugar, descargar el dataset y colocarlo a la misma altura de los archivos del proyecto

Aunque la red ya está entrenada, puede realizar nuevamente la selección de hiperparámetros y el entrenamiento.

Si se desea realizar  predicciones directamente, ejecutar el archivo Prediction.py donde aparecerá una interfaz visual para cargar las imágenes y predecir.

Por otro lado, si desea para realizar la selección de hiperparámetros, ejecutar el archivo SelectHyperparameters.py, tomará alrededor de 25 minutos, una vez terminado puede proceder a ejecutar el archivo Train.py para entrenar la red, tomará solo unos minutos. Finalmente puede pasar a ejecutar el archivo Prediction.py, tal como se mencionó previamente.
